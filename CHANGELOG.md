# Changelog

All notable changes will be documented in this file. The project follows the structure of [Keep a Changelog](https://keepachangelog.com/en/1.1.0/) and intends to use [Semantic Versioning](https://semver.org/) once releases begin.

## [Unreleased]

### Added

- Repository-inspecting agency prototype with persisted run data.
- Deterministic local demo and optional Hermes-backed generation path.
- Operator console for agency, catalog, operations, and observability views.
- File-backed X execution state machine with veto, block, ready, and receipt states.
- Safe browser handoff contract and executor self-test.
- Initial open-source documentation and community contribution templates.

### Known limitations

- No hosted service or tagged release exists.
- Hermes-backed generation and critic-driven revision are still being hardened.
- Notifications, scheduling, and recovery are not complete production workflows.
- The project has not yet demonstrated a repeatable set of verified public executions.
- Compatibility beyond the documented local toolchain has not been established.
